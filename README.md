# منصة الإحصائية الشهرية - V34

## الجديد في V34
- إضافة صفحة إدارة المراكز الصحية.
- إضافة مركز جديد من داخل النظام.
- تعديل بيانات المركز.

- تسجيل عمليات إدارة المراكز في سجل العمليات.
